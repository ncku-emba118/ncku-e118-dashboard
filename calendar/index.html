Frequently Asked 訂閱後常見問題
Q1. 訂閱後，活動什麼時候出現在我的手機？
通常 15-60 分鐘內手機行事曆會自動把活動拉下來。
如果你急著看到，可以打開手機行事曆 App 然後下拉刷新（iPhone：在月曆任一畫面下拉；Google Cal：點右上重新整理）。
Q2. 怎麼設定「活動前 N 分鐘提醒」？
提醒由你自己手機設定，幹部端不負責推送。打開手機行事曆 App → 點任一活動 → 編輯 → 加入提醒。
建議至少設兩種：「1 天前」+「1 小時前」。國定假日不需要提醒可以不設。
Q3. 退訂怎麼做？
iPhone：設定 → 行事曆 → 帳號 → 訂閱式行事曆 → 找到「E118 成大EMBA」→ 刪除帳號
Android / Google Calendar：Cal App → 設定 → 行事曆清單 → 找到 E118 → 取消勾選或移除
Q4. 換手機後還要重新訂閱嗎？
不用。訂閱跟著你的 iCloud / Google 帳號跑，新手機登入同一個帳號就會自動同步過來。
Q5. 為什麼我看到活動但沒收到提醒？
訂閱式行事曆的活動預設沒有提醒，必須你自己在手機上點該活動 → 加入提醒。
這是 iOS / Google Cal 的設計：每位同學對「想被打擾的時機」需求不同，所以提醒由訂閱者自己掌握。
Q6. 我能修改活動內容嗎？
不能。這本行事曆是唯讀訂閱，只有幹部（秘書長 / 班代 / 活動長 / 公關長）能在 Google Calendar 後台新增 / 編輯活動。
想新增活動：請通知對應幹部，由他們統一更新後 15-60 分鐘內你會看到。
